# BLDC_DRV

通用无刷电机矢量控制驱动板开源硬件

有F103RC和F405RC两个版本，布局几乎完全一致，支持霍尔、编码器、无感FOC和六步换向控制

使用ST MCSDK开发FOC控制程序，添加OLED中英文参数显示
